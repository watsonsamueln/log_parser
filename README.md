## Log Parser App
---
* First run:  `bundle install`
* then: `chmod +x ruby_app.rb`
* Run program via:
```./ruby_app.rb FILE/PATH/HERE.EXTENSION```
* (accepted file extensions: `.log` `.txt` `.rb`)
---
* To test, run: `rspec` in terminal


---

